Practice Bundle
===============

This directory tree is designed for command–line practice.

* **docs/**     — this file and potential documentation
* **logs/**     — assorted system and web logs
* **scripts/**  — simple Bash scripts (one intentionally has a bug)
* **data/**     — sample CSV data

Exercises you can try
---------------------

1. Find the three largest files under this tree.
2. Extract all requests that returned HTTP status 404 from logs/web_access.log.
3. Write a one–liner that counts failed SSH logins in logs/syslog.log per IP address.
4. Fix the typo in scripts/backup.sh, add execute permissions, and run it.
5. Import data/customers.csv into SQLite or MySQL and query the number of customers per city.

Happy hacking!
